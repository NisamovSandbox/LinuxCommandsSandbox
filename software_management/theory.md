=== Repositorios de Ubuntu / Debian

En sistemas basados en Ubuntu y Debian, los paquetes se distribuyen a través de **repositorios**, que son colecciones de paquetes agrupados según criterios de soporte, licencia y mantenimiento. Conocer estos repositorios es esencial para gestionar software de forma segura y eficiente.

Los repositorios principales son:

[table]
|===
| Repositorio | Descripción

| main
| Software libre y de código abierto, soportado oficialmente por Canonical. Incluye los paquetes esenciales para el sistema y garantiza soporte completo y actualizaciones de seguridad.

| restricted
| Software propietario necesario para el correcto funcionamiento del hardware, como controladores gráficos o de red. Incluye paquetes no libres pero esenciales para la compatibilidad del sistema.

| universe
| Software libre y de código abierto mantenido por la comunidad. No tiene soporte oficial de Canonical, pero es ampliamente usado y permite acceder a una mayor variedad de aplicaciones.

| multiverse
| Software propietario con restricciones legales o de licencia. Puede requerir la aceptación de términos específicos antes de su uso y no recibe soporte oficial.
|===
