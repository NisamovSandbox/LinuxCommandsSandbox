==== Descripción de Directorios

[.directory-listing]
* `/` (Raíz):: Directorio raíz del sistema de archivos. Todos los demás directorios y archivos se encuentran bajo este.
* `/bin`:: Programas ejecutables esenciales para el usuario.
* `/boot`:: Archivos de arranque del sistema.
* `/dev`:: Archivos especiales que representan dispositivos.
  ** `/dev/zero`:: Dispositivo especial de tipo carácter
  ** `/dev/null`:: Dispositivo especial de tipo carácter
* `/etc`:: Archivos de configuración del sistema y aplicaciones.
  ** `/etc/apt/sources.list.d/ubuntu.sources`:: Repositorios de software
  ** `/etc/network`:: Configuración de red
  ** `/etc/apache`:: Configuración del servidor Apache
  ** `/etc/nginx`:: Configuración del servidor Nginx
  ** `/etc/skel`:: Plantilla para nuevos usuarios
  ** `/etc/ssh`:: Configuración del protocolo SSH
* `/home`:: Directorios personales de los usuarios.
  ** `/home/user1/documents`:: Documentos del usuario1
  ** `/home/user1/downloads`:: Descargas del usuario1
  ** `/home/user2/documents`:: Documentos del usuario2
  ** `/home/user2/pictures`:: Imágenes del usuario2
* `/lib`:: Bibliotecas compartidas esenciales.
* `/mnt`:: Montaje de sistemas de archivos externos o temporales.
  ** `/mnt/backup`:: Punto de montaje para copias de seguridad
  ** `/mnt/data`:: Punto de montaje para datos adicionales
* `/opt`:: Paquetes de software adicionales.
* `/proc`:: Información sobre procesos y el sistema.
* `/root`:: Directorio personal del superusuario.
* `/run`:: Archivos de sistema durante el arranque.
* `/sbin`:: Programas esenciales para administradores.
* `/srv`:: Datos servidos por el sistema.
  ** `/srv/www`:: Datos de sitio web
  ** `/srv/ftp`:: Datos para servidor FTP
* `/sys`:: Interfaz del kernel.
* `/tmp`:: Archivos temporales.
* `/usr`:: Archivos y directorios para aplicaciones del sistema.
  ** `/usr/bin`:: Binarios de aplicaciones
  ** `/usr/lib`:: Bibliotecas
  ** `/usr/sbin`:: Binarios para administradores
  ** `/usr/share`:: Datos compartidos
  ** `/usr/src`:: Código fuente del kernel
* `/var`:: Datos variables (logs, correo, archivos temporales).
  ** `/var/log`:: Registros del sistema y aplicaciones
  ** `/var/mail`:: Buzones de correo
  ** `/var/www/html`:: Contenido HTML
  ** `/var/www/logs`:: Logs del servidor web
* `/media`:: Montaje de dispositivos extraíbles.
  ** `/media/cdrom`:: CD/DVD
  ** `/media/usb`:: USB
* `.`:: Apunta al directorio actual
* `..`:: Apunta al directorio superior
