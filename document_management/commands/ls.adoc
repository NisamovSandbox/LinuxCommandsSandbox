=== ls
Listar ficheros del directorio actual.
+
[unordered.stack]
-l:: `ls -l` Lista detallada
-a:: `ls -a` Mostrar ficheros ocultos
-h:: `ls -h` Mostrar lectura para humanos
-t:: `ls -t` Ordenar ficheros por fecha de modificación
-i:: `ls -i` Mostrar el número de inodo
-S:: `ls -S` Ordenar ficheros por tamaño (mayor a menor)
-r:: `ls -r` Ordenar ficheros invirtiendo el orden actual
-R:: `ls -R` Mostrar recursivamente los archivos
-d:: `ls -d` Mostrar unicamente directorios
-F:: `ls -F` Clasificar archivos según el tipo
-1:: `ls -1` Mostrar un fichero por linea
-Q:: `ls -Q` Mostrar los nombres de ficheros entre comillas
-v:: `ls -v` Ordenar naturalmente los ficheros con valor numérico
-X:: `ls -X` Ordenar ficheros por extensión
--color:: `ls --color` Agregar color a los diferente tipos de ficheros
--block-size:: `ls --block-size=<criterio>` Mostrar peso de los archivos
+
[unordered.stack]
b:: `--block-size=b` Mostrar peso por Bytes
k:: `--block-size=k` Mostrar peso por Kilobytes
M:: `--block-size=M` Mostrar peso por Megabytes
G:: `--block-size=G` Mostrar peso por Gigabytes
T:: `--block-size=T` Mostrar peso por Terabytes
P:: `--block-size=P` Mostrar peso por Petabytes
+
--time:: `ls --time=<valor>` Selecciona el campo de tiempo a usar
+
[unordered.stack]
-atime:: Tiempo de último acceso
-mtime:: Tiempo de última modificación (por defecto)
-ctime:: Tiempo de último cambio de estado
-birth:: Tiempo de creación (si está soportado)
+
--sort:: `ls --sort=<criterio>` Ordenar ficheros según un criterio
+
[unordered.stack]
size:: `ls -l --sort=size` Ordenar por peso
time:: `ls -l --sort=time` Ordenar por tiempo
version:: `ls -l --sort=version` Ordenar por versión
extension:: `ls -l --sort=extension` Ordenar por extensión
