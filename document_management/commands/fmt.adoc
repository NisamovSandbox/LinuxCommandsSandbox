=== fmt
Formatear texto ajustando líneas y párrafos para mejorar legibilidad.

[source,bash]
----
# Ajustar líneas a 75 caracteres
fmt -w 75 fichero.txt
----

[unordered.stack]
-w:: `fmt -w` Ancho máximo de línea

[source,bash]
----
# Ajustar líneas a 50 caracteres
fmt -w 50 fichero.txt
----

[unordered.stack]
--prefix:: `fmt --prefix` Mantener prefijo de cada línea

[source,bash]
----
# Mantener sangría de 4 espacios al formatear
fmt --prefix="    " fichero.txt
----

[unordered.stack]
-c:: `fmt -c` Centrar líneas

[source,bash]
----
# Centrar el texto (cuando se usa con ciertas configuraciones)
fmt -c fichero.txt
----

[unordered.stack]
--help:: `fmt --help` Mostrar ayuda

[source,bash]
----
# Ver opciones disponibles
fmt --help
----
