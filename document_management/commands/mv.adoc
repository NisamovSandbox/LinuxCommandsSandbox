=== mv
Mover archivo entre rutas.

[source,bash]
----
# Mover un fichero a otro directorio
mv archivo.txt /ruta/destino/
----

-i:: `mv -i` Solicitar confirmación antes de sobreescribir archivos

[source,bash]
----
# Mover pidiendo confirmación antes de sobrescribir
mv -i archivo.txt /ruta/destino/
----

-v:: `mv -v` Mostrar archivos de origen y destino (verbose)

[source,bash]
----
# Mover y mostrar información de la operación
mv -v archivo.txt /ruta/destino/
----

-f:: `mv -f` Forzar la acción (sobrescribir sin preguntar)

[source,bash]
----
# Mover forzando la sobreescritura
mv -f archivo.txt /ruta/destino/archivo.txt
----

-n:: `mv -n` No sobrescribir archivos existentes (no-clobber)

[source,bash]
----
# No sobrescribir si destino existe
mv -n archivo.txt /ruta/destino/
----

-b:: `mv -b` Hacer copia de seguridad del destino al sobrescribir

[source,bash]
----
# Usar copia de seguridad al mover
mv -b archivo.txt /ruta/destino/
----

--backup:: `mv --backup[=CONTROL]` Hacer copia de seguridad con control (simple/numbered/t) 

[source,bash]
----
# Mover con copia de seguridad numerada
mv --backup=numbered archivo.txt /ruta/destino/
----
