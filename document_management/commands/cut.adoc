=== cut
Extraer secciones de cada línea de un fichero o flujo de datos.

[source,bash]
----
# Uso básico: extraer campos o caracteres
cut -f 1 archivo.csv
----

* -c:: Extraer caracteres por rango

[source,bash]
----
# Extraer primeros 5 caracteres de cada línea
cut -c 1-5 fichero.txt
----

* -d:: `cut -d'<delimitador>'` Usar un delimitador para campos

[source,bash]
----
# Extraer la segunda columna usando ':' como separador
cut -d ':' -f 2 fichero.csv
----

* -f:: `cut -f <columna>` Seleccionar columnas (campos)

[source,bash]
----
# Extraer múltiples campos
cut -d ',' -f 1,3 datos.csv
----
