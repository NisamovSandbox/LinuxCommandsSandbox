=== locate
Buscar rápidamente archivos en el sistema usando la base de datos actualizada por `updatedb`.

[source,bash]
----
# Buscar archivos por nombre
locate fichero.txt
----

[unordered.stack]
-i:: `locate -i` Ignorar mayúsculas/minúsculas

[source,bash]
----
# Buscar sin diferenciar entre mayúsculas y minúsculas
locate -i FICHERO.TXT
----

[unordered.stack]
-r:: `locate -r` Usar expresiones regulares

[source,bash]
----
# Buscar todos los archivos que terminen en .log
locate -r '\.log$'
----

[unordered.stack]
-c:: `locate -c` Contar coincidencias

[source,bash]
----
# Contar cuántos archivos coinciden con el patrón
locate -c fichero.txt
----

[unordered.stack]
-l:: `locate -l` Limitar el número de resultados

[source,bash]
----
# Mostrar solo las primeras 10 coincidencias
locate -l 10 fichero.txt
----

[unordered.stack]
--existing:: `locate --existing` Mostrar solo archivos existentes

[source,bash]
----
# Filtrar resultados que aún existan
locate --existing fichero.txt
----

[unordered.stack]
--database:: `locate --database` Especificar base de datos

[source,bash]
----
# Usar base de datos personalizada
locate --database /tmp/mi_locate.db fichero.txt
----

[unordered.stack]
--help:: `locate --help` Mostrar ayuda

[source,bash]
----
# Ver opciones disponibles
locate --help
----
