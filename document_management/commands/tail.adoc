
=== tail
Mostrar las últimas líneas de uno o más ficheros.

[source,bash]
----
# Mostrar las últimas 10 líneas (por defecto)
tail fichero.txt
----

--lines, -n:: `tail -n <numero>` Mostrar las últimas N líneas

[source,bash]
----
tail -n 30 fichero.txt
----

-f:: `tail -f` Seguir un fichero conforme crece

[source,bash]
----
tail -f /var/log/syslog
----

-F:: `tail -F` Seguir fichero por nombre y reabrir si rota

[source,bash]
----
tail -F /var/log/syslog
----

-c:: `tail -c <bytes>` Mostrar por bytes en lugar de líneas

[source,bash]
----
tail -c 100 fichero.txt
----

-q:: `tail -q` Silenciar encabezados cuando se pasan varios ficheros

[source,bash]
----
tail -q fichero1.txt fichero2.txt
----
