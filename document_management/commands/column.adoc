=== column
Formatear la salida en columnas para mejorar la legibilidad.

[source,bash]
----
# Mostrar un listado en columnas
ls | column
----

* -t:: `column -t` Ajustar columnas en tabla

[source,bash]
----
# Separar por tabulaciones y alinear columnas
cat datos.txt | column -t
----

* -s:: `column -s` Definir delimitador

[source,bash]
----
# Usar coma como separador de columnas
cat datos.csv | column -s ',' -t
----

* -x:: `column -x` Llenar filas antes de columnas

[source,bash]
----
# Mostrar las columnas llenando filas primero
ls | column -x
----

* --help:: `column --help` Mostrar ayuda

[source,bash]
----
# Ver opciones disponibles
column --help
----
