=== scp
Copiar ficheros de forma segura entre hosts usando SSH.

[source,bash]
----
# Copiar un fichero a un host remoto
scp fichero.txt usuario@192.168.1.10:/home/usuario/
----

-r:: `scp -r` Copiar directorios recursivamente

[source,bash]
----
# Copiar un directorio recursivamente
scp -r proyecto/ usuario@servidor:/ruta/destino/
----

-P:: `scp -P <puerto>` Especificar puerto SSH

[source,bash]
----
# Copiar usando puerto 2222
scp -P 2222 fichero.txt usuario@servidor:/home/usuario/
----

-p:: `scp -p` Mantener permisos y fechas

[source,bash]
----
# Copiar manteniendo permisos/fechas
scp -p fichero.txt usuario@servidor:/home/usuario/
----

-C:: `scp -C` Comprimir datos durante la transferencia

[source,bash]
----
# Copiar con compresión
scp -C fichero.iso usuario@servidor:/home/usuario/
----

-i:: `scp -i <clave>` Usar archivo de clave privada para autenticar

[source,bash]
----
# Usar clave privada
scp -i ~/.ssh/id_rsa fichero.txt usuario@servidor:/home/usuario/
----

-B:: `scp -B` Modo batch (sin preguntas interactivas)

[source,bash]
----
# Usar en scripts sin interacción
scp -B fichero.txt usuario@servidor:/home/usuario/
----

-l:: `scp -l` Limitar ancho de banda (KB/s)

[source,bash]
----
# Limitar ancho de banda a 100KB/s
scp -l 100 fichero.txt usuario@servidor:/home/usuario/
----

-o:: `scp -o` Pasar opciones a SSH (ej: puerto, identidad)

[source,bash]
----
# Pasar opciones SSH (ejemplo: StrictHostKeyChecking=no)
scp -o StrictHostKeyChecking=no fichero.txt usuario@servidor:/home/usuario/
----
