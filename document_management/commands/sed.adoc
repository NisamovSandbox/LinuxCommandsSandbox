=== sed
Editor de flujo de texto para filtrar y transformar texto línea a línea.

[source,bash]
----
# Reemplazo simple en un archivo
sed 's/viejo/nuevo/' fichero.txt
----

-i:: `sed -i` Modificar el fichero directamente

[source,bash]
----
# Reemplazar “viejo” por “nuevo” en el archivo original
sed -i 's/viejo/nuevo/g' fichero.txt
----

-e:: `sed -e` Ejecutar múltiples comandos

[source,bash]
----
# Reemplazar y eliminar líneas al mismo tiempo
sed -e 's/viejo/nuevo/g' -e '/^#/d' fichero.txt
----

-n:: `sed -n` Suprimir salida automática

[source,bash]
----
# Mostrar solo líneas que coincidan con el patrón
sed -n '/error/p' fichero.log
----

-d:: `sed 'Nd'` Eliminar líneas específicas

[source,bash]
----
# Eliminar la tercera línea
sed '3d' fichero.txt
----

-a:: `sed 'Nq'` Imprimir hasta cierta línea

[source,bash]
----
# Imprimir solo las primeras 5 líneas
sed -n '1,5p' fichero.txt
----

-r:: `sed -r` Expresiones regulares extendidas

[source,bash]
----
# Usar regex extendido para reemplazo
sed -r 's/(foo|bar)/baz/g' fichero.txt
----

--follow-symlinks:: `sed --follow-symlinks` Modificar enlaces simbólicos

[source,bash]
----
# Editar fichero que es un enlace simbólico
sed -i --follow-symlinks 's/viejo/nuevo/' enlace.txt
----
