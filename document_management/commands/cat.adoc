=== cat
Concatenar y mostrar el contenido de ficheros.

[source,bash]
----
# Mostrar un fichero
cat fichero.txt
----

-n:: `cat -n` Numerar líneas de salida

[source,bash]
----
# Mostrar con numeración de líneas
cat -n fichero.txt
----

-b:: `cat -b` Numerar sólo líneas no vacías

[source,bash]
----
# Numerar sólo líneas no vacías
cat -b fichero.txt
----

-A:: `cat -A` Mostrar caracteres especiales

[source,bash]
----
# Mostrar caracteres especiales (tabuladores, finales de línea)
cat -A fichero.txt
----
