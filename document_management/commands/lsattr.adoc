=== lsattr
Mostrar los atributos de los archivos en sistemas de ficheros ext2/ext3/ext4.

[source,bash]
----
# Mostrar atributos de un archivo
lsattr fichero.txt
----

[unordered.stack]
-R:: `lsattr -R` Recursivo

[source,bash]
----
# Mostrar atributos de todos los archivos en un directorio recursivamente
lsattr -R /ruta/directorio
----

[unordered.stack]
-d:: `lsattr -d` Mostrar atributos de directorios en lugar de su contenido

[source,bash]
----
# Mostrar atributos de un directorio
lsattr -d /ruta/directorio
----

[unordered.stack]
-V:: `lsattr -V` Mostrar versión del comando

[source,bash]
----
# Ver versión de lsattr
lsattr -V
----

[unordered.stack]
--help:: `lsattr --help` Mostrar ayuda

[source,bash]
----
# Ver opciones disponibles
lsattr --help
----

[unordered.stack]
--quiet:: `lsattr --quiet` Suprimir mensajes innecesarios

[source,bash]
----
# Mostrar solo información esencial
lsattr --quiet fichero.txt
----
