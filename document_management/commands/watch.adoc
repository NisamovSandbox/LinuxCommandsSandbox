=== watch
Ejecutar un comando periódicamente y mostrar su salida en pantalla.

[source,bash]
----
# Ejecutar un comando cada 2 segundos
watch ls
----


-n:: `watch -n` Intervalo de actualización

[source,bash]
----
# Ejecutar cada 5 segundos
watch -n 5 date
----

-d:: `watch -d` Resaltar cambios entre ejecuciones

[source,bash]
----
# Mostrar diferencias entre actualizaciones
watch -d free -h
----

-t:: `watch -t` Ocultar cabecera

[source,bash]
----
# Mostrar solo la salida del comando
watch -t uptime
----

-g:: `watch -g` Salir cuando cambie la salida

[source,bash]
----
# Finalizar al detectar un cambio
watch -g cat estado.txt
----

-e:: `watch -e` Detener en caso de error

[source,bash]
----
# Parar si el comando devuelve error
watch -e ls /ruta/inexistente
----

-b:: `watch -b` Resaltar cambios con negrita

[source,bash]
----
# Mostrar cambios en negrita
watch -b df -h
----

-quoted:: Uso con comillas

[source,bash]
----
# Ejecutar comandos con tuberías
watch 'ps aux | grep nginx'
----
