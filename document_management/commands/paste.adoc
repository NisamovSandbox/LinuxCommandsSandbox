=== paste
Combinar columnas de uno o más ficheros línea a línea.

[source,bash]
----
# Combinar dos archivos separando por tabulador
paste fichero1.txt fichero2.txt
----

-d:: `paste -d` Definir delimitador

[source,bash]
----
# Usar coma como separador
paste -d ',' fichero1.txt fichero2.txt
----

-s:: `paste -s` Serializar líneas de un fichero

[source,bash]
----
# Combinar todas las líneas de un archivo en una sola
paste -s fichero.txt
----

--help:: `paste --help` Mostrar ayuda

[source,bash]
----
# Ver opciones disponibles
paste --help
----

--version:: `paste --version` Mostrar versión

[source,bash]
----
# Ver versión del comando
paste --version
----

-m:: `paste -m` Combinar múltiples archivos de forma paralela

[source,bash]
----
# Combinar ficheros mezclando sus líneas
paste -m fichero1.txt fichero2.txt
----
