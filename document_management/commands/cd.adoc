=== cd
Establecer directorio de trabajo.

[source,bash]
----
# Ir al directorio padre
cd ..
# Ir a una ruta absoluta
cd /home/usuario/Documentos
----
[unordered.stack]
-~:: `cd ~` Ir al directorio home del usuario

[source,bash]
----
# Volver al directorio anterior (OLDPWD)
cd ~
----
[unordered.stack]
-..:: `cd ..` Acceder al directorio padre

[source,bash]
----
cd ..
----
[unordered.stack]
--:: `cd -` Volver al directorio anterior (OLDPWD)

[source,bash]
----
# Volver al directorio anterior (OLDPWD)
cd -
----
