=== find
Buscar ficheros y directorios que cumplan criterios especificados.

[source,bash]
----
# Buscar por nombre en árbol actual
find . -name "*.txt"
----

* -name:: `find -name` Buscar por nombre

[source,bash]
----
# Buscar archivos con extensión .log
find /var/log -name "*.log"
----

* -type:: `find -type` Buscar por tipo (`f` fichero, `d` directorio)

[source,bash]
----
# Buscar sólo ficheros
find . -type f -name "*.conf"
----

* -mtime:: `find -mtime` Buscar por fecha de modificación

[source,bash]
----
# Buscar ficheros modificados en los últimos 7 días
find . -mtime -7
----

* -atime:: `find -atime` Buscar por fecha de acceso

[source,bash]
----
# Buscar archivos accedidos en el último día
find . -atime -1
----

* -size:: `find -size` Buscar por tamaño (ej: 1M)

[source,bash]
----
# Buscar ficheros mayores a 1M
find /ruta -size 1M
----

* -maxdepth:: `find -maxdepth` Limitar profundidad máxima

[source,bash]
----
# Limitar búsqueda a profundidad 2
find . -maxdepth 2 -name "*.sh"
----

* -mindepth:: `find -mindepth` Limitar profundidad mínima

[source,bash]
----
# Buscar a partir de profundidad 1
find . -mindepth 1 -name "README.md"
----

* -exec:: `find -exec` Ejecutar comando sobre resultados

[source,bash]
----
# Cambiar permisos de los ficheros encontrados
find . -type f -exec chmod 644 {} \;
----
