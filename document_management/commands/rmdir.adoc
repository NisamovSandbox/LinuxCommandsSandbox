=== rmdir
Eliminar directorios vacíos.

[source,bash]
----
# Eliminar un directorio vacio
rmdir Documentos
----

- (uso):: `rmdir <directorio>` Eliminar directorio vacío

[source,bash]
----
# Eliminar un directorio vacío
rmdir Documentos

# Intentar eliminar un directorio no vacío falla
rmdir carpeta_con_contenido || echo "El directorio no está vacío"
----
