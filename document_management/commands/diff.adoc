=== diff
Comparar ficheros línea por línea y mostrar diferencias.

[source,bash]
----
# Comparar dos ficheros
diff fichero1.txt fichero2.txt
----

* -u:: `diff -u` Mostrar diferencias en formato unificado

[source,bash]
----
# Comparar en formato unificado (útil para parches)
diff -u fichero1.txt fichero2.txt > cambios.patch
----

* -c:: `diff -c` Mostrar diferencias con contexto

[source,bash]
----
# Comparar mostrando contexto adicional
diff -c fichero1.txt fichero2.txt
----

* -r:: `diff -r` Comparar directorios recursivamente

[source,bash]
----
# Comparar dos directorios recursivamente
diff -r dir1/ dir2/
----

* -i:: `diff -i` Ignorar mayúsculas/minúsculas

[source,bash]
----
# Ignorar case
diff -i fichero1.txt fichero2.txt
----
