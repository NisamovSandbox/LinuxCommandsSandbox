=== tar
Empaquetamiento de múltiples archivos y directorios.

TAR (Tape ARchive) es un software que empaqueta múltiples archivos y directorios en un único archivo contenedor (`.tar`). 
**Importante**: TAR por sí solo no comprime, solo empaqueta.

[source,bash]
----
# Crear un tar simple
tar -cf archivo.tar archivo1 archivo2
----
[unordered.stack]
-c:: `tar -c` Crear un nuevo archivo tar

[source,bash]
----
# Crear archivo tar
tar -cf proyecto.tar src/
----
[unordered.stack]
-f:: `tar -f` Especificar nombre de archivo tar

[source,bash]
----
# Especificar archivo con -f
tar -cf archivo.tar carpeta/
----
[unordered.stack]
-x:: `tar -x` Extraer archivos de un tar

[source,bash]
----
# Extraer archivo tar
tar -xf archivo.tar
----
[unordered.stack]
-t:: `tar -t` Listar contenido del tar

[source,bash]
----
# Listar contenido
tar -tf archivo.tar
----
[unordered.stack]
-v:: `tar -v` Verbose: mostrar archivos procesados

[source,bash]
----
# Crear con verbose
tar -cvf archivo.tar carpeta/
----
[unordered.stack]
-z:: `tar -z` Comprimir/Descomprimir usando gzip (con -c/-x)

[source,bash]
----
# Comprimir con gzip
tar -czf proyecto.tar.gz carpeta/

# Descomprimir gzip
tar -xzf proyecto.tar.gz
----
[unordered.stack]
-j:: `tar -j` Usar bzip2

[source,bash]
----
# Comprimir con bzip2
tar -cjf proyecto.tar.bz2 carpeta/

# Descomprimir bzip2
tar -xjf proyecto.tar.bz2
----
[unordered.stack]
-J:: `tar -J` Usar xz

[source,bash]
----
# Comprimir con xz
tar -cJf proyecto.tar.xz carpeta/

# Descomprimir xz
tar -xJf proyecto.tar.xz
----
[unordered.stack]
--exclude:: `tar --exclude=<patrón>` Excluir archivos/paths coincidentes

[source,bash]
----
# Crear tar excluyendo archivos .git
tar -czf proyecto.tar.gz --exclude='.git' carpeta/
----
[unordered.stack]
--strip-components:: `tar --strip-components=N` Omitir N componentes del path al extraer

[source,bash]
----
# Extraer omitiendo los dos primeros componentes del path
tar -xzf proyecto.tar.gz --strip-components=2
----
[unordered.stack]
--no-same-owner:: `tar --no-same-owner` No restaurar propietario (útil como usuario no root)

[source,bash]
----
# Extraer sin intentar restaurar propietarios
tar --no-same-owner -xzf proyecto.tar.gz
----
[unordered.stack]
--remove-files:: `tar --remove-files` Eliminar archivos que se han archivado

[source,bash]
----
# Añadir archivos al tar y borrarlos del origen
tar -cvf archivos.tar --remove-files *.log
----
