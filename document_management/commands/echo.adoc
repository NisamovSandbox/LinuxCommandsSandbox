=== echo
Mostrar una línea de texto o variables en la salida estándar.

[source,bash]
----
# Mostrar una cadena simple
echo "Hola Mundo"
----

* -n:: `echo -n` No añadir salto de línea final

[source,bash]
----
# No añadir salto de línea
echo -n "Hola"
----

* -e:: `echo -e` Interpretar escapes como \n, \t

[source,bash]
----
# Interpretar escapes
echo -e "Linea1\nLinea2"
----
