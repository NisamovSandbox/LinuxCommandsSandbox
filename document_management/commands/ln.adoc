=== ln
Creación de enlaces duros y simbólicos.

[source,bash]
----
# Crear un enlace simbólico
ln -s /ruta/origen/archivo.txt enlace.txt
----

* -s:: `ln -s` Crear un enlace simbólico

[source,bash]
----
# Crear enlace simbólico
ln -s /ruta/origen/archivo.txt enlace.txt
----

* -f:: `ln -f` Forzar creación de enlace sobrescribiendo el destino

[source,bash]
----
# Forzar creación de enlace y sobrescribir si existe
ln -f archivo.txt enlace_existente.txt
----

* -i:: `ln -i` Solicitar confirmación antes de sobrescribir

[source,bash]
----
# Crear enlace solicitando confirmación
ln -i archivo.txt enlace.txt
----

* --help:: `ln --help` Mostrar ayuda breve

[source,bash]
----
ln --help
----
