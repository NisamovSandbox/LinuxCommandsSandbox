=== setfacl
Establecer o modificar listas de control de acceso (ACL) en ficheros y directorios.

[source,bash]
----
# Asignar permisos ACL básicos
setfacl -m u:usuario:rwx fichero.txt
----

-m:: `setfacl -m` Modificar o añadir entradas ACL
+
[source,bash]
----
# Permitir lectura y ejecución a usuario “juan”
setfacl -m u:juan:rx programa.sh
----

-x:: `setfacl -x` Eliminar entradas ACL
+
[source,bash]
----
# Eliminar permiso específico de usuario
setfacl -x u:ana fichero.txt
----

-b:: `setfacl -b` Quitar todas las ACL
+
[source,bash]
----
# Eliminar todas las ACL, dejando permisos estándar
setfacl -b fichero.txt
----

-k:: `setfacl -k` Quitar la máscara (mask)
+
[source,bash]
----
# Eliminar la máscara ACL
setfacl -k fichero.txt
----

-R:: `setfacl -R` Aplicar recursivamente
+
[source,bash]
----
# Asignar permisos a todos los ficheros de un directorio
setfacl -R -m g:grupo:rw /ruta/directorio
----

-d:: `setfacl -d` Establecer ACL por defecto
+
[source,bash]
----
# Definir permisos por defecto para nuevos ficheros
setfacl -d -m u:guest:rx /ruta/directorio
----

--remove-all:: `setfacl --remove-all` Eliminar todas las ACL (equivale a -b)
+
[source,bash]
----
# Eliminar todas las ACL explícitamente
setfacl --remove-all fichero.txt
----
